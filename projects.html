SentinelNet — Enterprise Full-Stack SOC Platform (In Progress)

SentinelNet is an enterprise-level, full-stack Security Operations Center (SOC) platform designed to centralize security visibility, alerting, and response workflows. The infrastructure is built and managed as code using AWS CDK, with API-driven services to support scalable ingestion and automation.

  • Building cloud infrastructure as code with AWS CDK to support production-style deployment and repeatability.
  • Developing API-based backend services for ingesting, storing, and querying security-relevant events.
  • Designing the platform with SOC workflows in mind (triage, enrichment, alert routing, and operational visibility).

Tech: AWS CDK, AWS (cloud services), APIs, full-stack architecture
